#+title: Chess

 Simple chess client for the terminal.

* Building

Clone the repository and build the project using =make=.

#+begin_src bash
git clone https://github.com/8dcc/chess
cd chess
make
# ...
#+end_src

* Usage

For more information about the program, run it with the =--help= argument.

#+begin_src bash
./chess-ncurses --help
# ...
#+end_src
